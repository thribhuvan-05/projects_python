# Website Automation Bot

This project automates browser actions such as opening websites and filling login forms automatically.

It demonstrates browser automation using Python.

## Features
- Opens websites automatically
- Fills login forms
- Simulates user interaction
- Automates repetitive tasks

## Technologies Used
- Python
- Selenium

## Installation

Install Selenium:

pip install selenium

Download ChromeDriver and ensure it matches your Chrome version.

## Usage

Run the program:

python automation_bot.py

The bot will open the browser and perform automated actions.
